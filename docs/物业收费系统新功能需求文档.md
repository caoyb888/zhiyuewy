## 1. 系统概述

本系统面向物业管理公司，提供覆盖B端（企业内部管理）与C端（业主服务）的一体化物业收费解决方案。B端管理平台支撑公司组织架构、项目档案、费用定义、抄表计费、收银对账、数据报表等核心业务；C端基于微信公众号/小程序，为业主提供账单查询、在线缴费、电子收据、信息推送、管家联络等便民服务。

## 2. B端管理平台功能需求

### 2.1 收费模块-看板

| 功能项 | 功能描述 |
|--------|----------|
| 数据看板 | 面向公司领导的小区数据信息整体管控视图，涵盖收费、工单、设备、业主活跃等数据分析 |
| 定时推送 | 每天晚上18:00，自动将项目收费情况、收费方式、收缴率分析、欠费情况等数据推送至领导层手机 |

### 2.2 企业信息

| 功能项 | 功能描述 |
|--------|----------|
| 公司管理 | 设定公司组织架构，支持总公司、分公司及下属管理处层级配置 |
| 组织管理 | 根据公司具体部门设定部门及部门下设职位 |
| 角色管理 | 根据不同职位分配相应的软件操作权限 |
| 人员管理 | 员工账号管理，包括用户名、密码、权限及小区分配权限设定 |
| 小区公告 | 在公司内部发送通知，所有操作账号可收到消息推送 |

### 2.3 基础档案

| 功能项 | 功能描述 |
|--------|----------|
| 项目管理 | 管理项目具体地址、项目联系电话、项目负责人等相关信息 |
| 房间管理 | 管理房间面积、编号、业主名称、所在楼层单元、附属车库及储藏室信息 |
| 客户管理 | 记录业主姓名、联系电话，并将业主名与资产进行绑定 |
| 仪表管理 | 管理房产中仪表的仪表编号、类型、安装时间等信息 |
| 车场管理 | 管理车场区域编号、车位区域划分 |
| 车位管理 | 管理车位编号、车位所属区域、车位相应的产权信息 |

### 2.4 物业收费

| 功能项 | 功能描述 |
|--------|----------|
| 费用定义 | 针对项目所有收费类型进行定义，支持自定义单价、公式、滞纳金方式、四舍五入方式、专款专冲方式、发票设置、税率设置等 |
| 费用分配 | 将费用定义中的费用分配到每个房间，支持房间收费个性化与多样化配置 |
| 票据管理 | 管理收款单/缴费通知单起始编码及张数；设置标题、收款单位、纸张格式、上传公章与Logo、收据备注、表单字段自定义及字段排序 |
| 抄表管理 | 支持每次抄表录数、仪表数据导入、损耗设置、公摊设置、倍率设置、单户入账、批量入账、自动计算、自动保存、优惠操作、预警检测 |
| 生成应收 | 批量生成或删除周期类费用，支持按月重新生成 |
| 生成应收临时 | 批量生成临时类费用 |
| 收银台 | 支持按房间简称、住户名称、树状图展开进入房间；勾选费用后按费用名称、年份、费用类型筛选；提供优惠操作、专款专冲、减免违约金、收据备注、收款操作；包含应收管理、单户生成费用、生成临时费用、预收款台账、押金台账退还押金、收款记录、房屋过户、提醒功能 |
| 折扣管理 | 设置优惠折扣名称及优惠比例 |
| 换发票 | 支持通过收据号查询并记录发票号码 |
| 批量冲预收款 | 筛选出费用后进行批量冲抵操作 |

### 2.5 数据查询

| 功能项 | 功能描述 |
|--------|----------|
| 交易汇总 | 按费用名称及收款方式统计已收费用，用于财务对账 |
| 交易明细 | 按票据号统计已收费用，展示收费时间、收款人、收款方式、费用金额 |
| 收费率报表 | 按月份、费用、账单月、收费时间等统计应收、实收、收本期、收清欠、收预收、收费率等信息 |
| 欠费明细表 | 按账单月等筛选条件统计各房间欠费明细 |
| 账单通知 | 按账单月等筛选条件统计各房间欠费，支持打印缴费通知单 |
| 应收管理 | 统计应收金额，支持删除未收费用、费用补录 |
| 综合查询表 | 通过各条件组合灵活查询所需数据统计 |
| 交易记录 | 统计已收收据、撤销收款、冲红、复核收据、撤销复核等操作记录 |
| 收费明细报表 | 按筛选条件统计收费明细 |
| 折扣明细表 | 按账单月、缴费时间等筛选条件统计费用优惠情况 |
| 预收款报表 | 按缴费时间等筛选条件统计预收款冲抵明细 |
| 收清欠汇总表 | 按收费时间统计收清欠费用情况 |
| 清欠率 | 按筛选条件统计欠收金额、收清欠金额并计算清欠率 |
| 过户查询 | 按时间段统计过户情况 |
| 提醒明细 | 统计各房间内添加的提醒内容 |
| 费用情况表 | 通过颜色区分每账单月未生成费用、已生成费用、已缴费情况 |
| 发票统计表 | 根据筛选条件查询开具发票的信息 |
| 押金记录表 | 按收款日期统计装修押金收取及退还情况 |
| 日结明细表 | 按账单月份与收费时间统计日结数据 |
| 调账记录 | 按时间段查询软件中进行过调账的数据 |
| 银行信托 | 支持通过银行进行费用的代收 |

## 3. C端业主微信端功能需求

### 3.1 通知公告

| 功能项 | 功能描述 |
|--------|----------|
| 通知公告 | 适用于物业集中向外界发布公司内部信息，如广告、通知、活动等内容 |

### 3.2 微信缴费

| 功能项 | 功能描述 |
|--------|----------|
| 费用查缴 | 便于业主在线查询欠费账单明细及历史缴费记录；无需去现场，通过公众号即可在线缴费，钱款直接进入对公账户，保证资金安全，提高项目收费率 |
| 电子收据 | 缴费成功后可以获取电子收据，便于留存缴费凭证 |
| 欠费通知 | 后台一键发送欠费明细账单，业主可在线查看 |

### 3.3 信息精准推送

| 功能项 | 功能描述 |
|--------|----------|
| 信息推送 | 实现精准/模糊推送，节日祝福、各种通知可实时传达给业主，提升物业服务品质 |

### 3.4 管家展示与评价

| 功能项 | 功能描述 |
|--------|----------|
| 管家信息 | 通过后台上传管家照片、网格管理、联系电话等信息，公众号界面显示业主专属管家信息并提供一键拨打电话功能 |

### 3.5 便民信息

| 功能项 | 功能描述 |
|--------|----------|
| 便民信息 | 物业公司后台进行项目周边各类服务商电话维护，便于业主查询使用（例：汽车站、火车站、热力公司、送水、快递、其他服务商等电话） |
